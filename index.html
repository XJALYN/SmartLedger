SmartLedger 是一款 AI 驱动的 iOS 智能记账应用。支持文字、语音或小票拍照输入， AI 自动提取消费明细，确认后保存至可搜索账本，并提供多维度支出统计。

查看源码 阅读 README

AI Assistant Chat · 自然语言 / 语音 / 小票识别

核心功能

从输入到确认、账本到统计，完整覆盖日常记账流程

💬

AI 聊天记账

自然语言记账与消费查询，支持语音输入和小票拍照，接入 DashScope Qwen 模型。

✅

确认并保存

可编辑 AI 提取的标题、金额、分类、商户和备注，保存后聊天卡片同步显示已保存状态。

📒

支出账本

按日分组展示，支持搜索与分类筛选，左滑删除，数据本地安全存储。

📊

消费统计

周 / 月 / 年 / 自定义范围，分类环形图、每日柱状图与预算进度一目了然。

🎨

主题与语言

5 种主题色，多语言支持（中 / 英 / 日 / 德 / 法 / 西），多货币与 CSV / JSON / PDF 导出。

⚡

算力积分

StoreKit 2 内购充值 AI 算力，未配置 API Key 时可使用离线 mock 解析。

应用截图

SwiftUI 实现，界面风格与产品原型保持一致
